# Hand Gesture Recognition

A real-time Hand Gesture Recognition project using Python, OpenCV, and MediaPipe.

## Features
- Real-time hand tracking using webcam
- Recognizes multiple hand gestures
- Detects:
  - Palm ✋
  - Thumb 👍
  - Fist ✊
  - Index ☝️
  - L Gesture

## Technologies Used
- Python
- OpenCV
- MediaPipe
- TensorFlow
